## 📖 Overview

This repository serves as a practical showcase and development ground for various cybersecurity tools implemented in Python. It includes a range of utilities designed to demonstrate fundamental concepts and techniques in areas such as cryptography and system monitoring. Each project within this repository is self-contained, allowing for easy exploration and focused learning.

## ✨ Features

The repository currently includes the following tools:

-   **Caesar Cipher**: An implementation of the classic Caesar cipher algorithm for basic text encryption and decryption. This tool helps understand foundational cryptographic principles.
-   **Keyloggers**: A directory containing implementations of keylogging functionalities. These tools are designed for educational purposes to demonstrate how keyboard inputs can be captured and logged, highlighting important security implications.

## 🚀 Quick Start

### Prerequisites

Ensure you have Python 3.11 installed on your system.

-   [Python 3.11](https://www.python.org/downloads/)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/alexl-vt28/Cybersecurity-Projects.git
    cd Cybersecurity-Projects
    ```

2.  **Install dependencies**
    Install the necessary Python packages using `pip` from the `requirements.txt` file:
    ```bash
    pip install -r requirements.txt
    ```
